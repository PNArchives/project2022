### Test Case ID
TES-102

### Test Case Name
ステータス

### Summary
ステータスを使ってテストケースを検索できるかを確認

### Precondition
* テストケースの一覧

### Steps

#### Action
「テスト仕様管理画面」を開いているかを確認する
#### Result
「テスト仕様管理画面」が表示されている。そうでなければホーム画面から移動する

#### Action
画面左上の「フィルタ」にあるステータスを選択するプルダウンより、任意のステータスを選ぶ
#### Result
画面左下の「フィルタ結果」にて、Actionで選んだステータスを持つテストケースのみが表示される
