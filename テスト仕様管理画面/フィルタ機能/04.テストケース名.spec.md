### Test Case ID
TES-101

### Test Case Name
テストケース名

### Summary
テストケース名を使ってテストケースを検索できるかを確認

### Precondition
* テストケースの一覧

### Steps

#### Action
「テスト仕様管理画面」を開いているかを確認する
#### Result
「テスト仕様管理画面」が表示されている。そうでなければホーム画面から移動する

#### Action
画面左上の「フィルタ」にあるテストケース名の入力欄に任意の文字を入力し、Enterキーを押す
#### Result
画面左下の「フィルタ結果」にて、テストケース名の中に、Actionで入力した文字が存在するテストケースのみが表示される
