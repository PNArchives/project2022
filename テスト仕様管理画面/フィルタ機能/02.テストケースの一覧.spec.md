### Test Case ID
TES-99

### Test Case Name
テストケースの一覧

### Summary
選択されているプロジェクトに存在するテストスイート、テストケースの一覧がツリービュー形式で表示される

### Precondition
* 環境の準備
* プロジェクトが選択されている
* 選択されたプロジェクト内にテストケースが存在する

### Steps

#### Action
ホーム画面にある「テスト仕様管理」のボタンをクリックする
#### Result
「テスト仕様管理画面」が表示される

#### Action
画面左下の「フィルタ結果」の中身を確認する
#### Result
選択されているプロジェクトのテストスイート、テストケースの一覧がツリービュー形式で表示される
