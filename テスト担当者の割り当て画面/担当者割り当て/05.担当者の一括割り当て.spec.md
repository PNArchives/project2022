### Test Case Name
担当者の一括割り当て

### Summary
 1件のテストケースに担当者を割り当てる

### Precondition
* プロジェクトが選択されている
* テスト計画が選択されている
* ビルドが選択されている
* 選択されたテスト計画、ビルドに紐付いているテストケースが複数存在する

### Steps

#### Action
対象のユーザーをプルダウンから選択する
#### Result
ユーザが選択される

#### Action
対象のテストケースのチェックボックスにチェックを入れて、割り当てボタンを押す
#### Result
チェックを入れたテストケースの担当者が選択したユーザーになる
