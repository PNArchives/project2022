### Test Case ID
TES-73

### Test Case Name
テスト計画選択

### Summary
テスト仕様に担当者を割り当てる対象となるテスト計画を選択する

### Precondition
* プロジェクトが選択されている
* 選択されたプロジェクト内に1つ以上のテスト計画が存在する

### Steps

#### Action
「テスト計画を選択」プルダウンをクリックする
#### Result
テスト計画の一覧がプルダウンに表示される

#### Action
テスト計画の一覧から一つ選択する
#### Result
クリックしたテスト計画が選択される
